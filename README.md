# SC1003 End-of-Year Project

This repository contains my **SC1003 end-of-year Python project**, completed as part of NTU coursework. The project focuses on grouping students into classes based on grade-related metrics, with the aim of creating a more balanced distribution of academic performance across classes.

## Project Overview

The program takes in student grade data and groups students into different classes using a simple balancing approach. The goal is to reduce large differences in average grades between classes.

This project was mainly an exercise in applying programming concepts to a structured problem, including:

- Working with data in Python  
- Implementing basic grouping and sorting logic  
- Thinking about fairness and distribution  
- Structuring a small Python program  

## Features

- Reads and processes student grade data  
- Groups students into a specified number of classes  
- Attempts to balance class averages  
- Outputs class assignments and summary statistics  

## Approach

The grouping approach uses basic sorting and assignment logic to distribute students across classes. The focus was on producing reasonably balanced results rather than optimizing for a perfect solution.

## Tech Stack

- Python 3  
- Standard Python libraries
